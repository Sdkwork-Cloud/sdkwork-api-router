# video/openai

## Live Mirror Family

- OpenAPI tag: `video.openai`
- Public path family: `/v1/videos*`
- Status: active

## Primary Routes

- `GET /v1/videos`
- `POST /v1/videos`
- `GET /v1/videos/{video_id}`
- `GET /v1/videos/{video_id}/content`
- `POST /v1/videos/{video_id}/remix`
- `POST /v1/videos/edits`
- `POST /v1/videos/extensions`
- `POST /v1/videos/{video_id}/extend`

## Notes

- Sora 2 and Sora 2 Pro stay in this shared official video family.
